# Drop Number Game

Java Swing ve Çoklu Bağlantılı Liste (Multi-Linked List) yapısı kullanılarak geliştirilmiş arayüzlü bir sayı düşürme oyunudur.

- **Multi-Linked List:** Her hücre (`NisaAktasNode`) `up`, `down`, `next` ve `prev` işaretçileri ile 4 yönlü bir zincire bağlıdır.
- **Senaryo Odaklı:** Oyun, önceden tanımlanmış özel bir sayı ve sütun senaryosuna göre adım adım ilerler.
- **Dinamik Birleşme:** Aynı sütuna düşen aynı değerli sayılar dikey olarak üst üste birleşerek iki katına katlanır.
